# Samaritan

Features:
* Voice based navigation
* Aadhar Card Reader
* Movie Recommendations and Details
* NFC enabled
* Auto silencer

Stuff used:
* [OMDB API](https://www.omdbapi.com)
* Google Voice Engine
* Recycler and Card Design
* [Realm DB](https://realm.io)
* [ZXingBarcodeScanner](https://www.github.com/zxing/zxing)
